## 技术细节

### 1. 均线计算方式
- **算法**：简单移动平均（SMA）
- **公式**：MA(n) = (P1 + P2 + ... + Pn) / n
- **数据源**：收盘价（Close）

### 2. 显示条件
- **MA7**：需要≥7个数据点（启动即可显示）
- **MA25**：需要≥25个数据点（启动即可显示）
- **MA60**：需要≥60个数据点（启动即可显示，因为获取了120根历史K线）
- **MA99**：需要≥99个数据点（启动即可显示）

### 3. 更新频率
- **分时图（1m）**：每500ms更新一次
- **K线图（5m/1h/1d）**：随K线数据更新而更新
- 所有均线都会随价格数据实时重新计算

### 4. 代码位置
**文件**: `src/finplot/plot.py`

#### 分时图均线（第136-142行）

#### K线图均线（第162-175行）
```python
# === 添加均线（K线图） ===
# 为5m、1h、1d图表添加MA5、MA7、MA99三条均线
ma_configs = [
    (5, "#FF6B6B", "MA5"),    # 红色 - 5周期均线
    (7, "#4ECDC4", "MA7"),    # 青色 - 7周期均线
    (99, "#A8E6CF", "MA99"),  # 绿色 - 99周期均线
]

for period, color, label in ma_configs:
    if len(df) >= period:
        ma = df["Close"].rolling(window=period).mean()
        vars.plots[key + f" ma{period}"] = fplt.plot(
            ma, ax=ax, color=color, width=1.5, legend=label
        )
```

#### 实时更新逻辑（第335-344行）
```python
# 更新均线数据（用于所有图表）
if plot_type.startswith("ma"):
    try:
        period = int(plot_type[2:])
        if len(df) >= period:
            ma = df["Close"].rolling(window=period).mean()
            vars.plots[key].update_data(ma)
    except (ValueError, KeyError):
        pass  # 忽略无效的均线key
```

## 均线交易含义

### MA5（5周期均线）
- **含义**：短期平均价格
- **用途**：
  - 价格突破MA5向上 → 短期看涨信号
  - 价格跌破MA5向下 → 短期看跌信号

### MA7（7周期均线）
- **含义**：一周（7日）平均价格
- **用途**：
  - 确认短期趋势
  - MA5与MA7形成黄金交叉（MA5上穿MA7）→ 买入信号
  - MA5与MA7形成死亡交叉（MA5下穿MA7）→ 卖出信号

### MA60（60周期均线）
- **含义**：1小时平均价格（用于1分钟图）
- **用途**：
  - 判断日内中长期趋势
  - 重要支撑/阻力位参考

### MA99（99周期均线）
- **含义**：长期平均价格
- **用途**：
  - 判断大趋势方向
  - 强支撑/阻力位
  - 价格在MA99之上为牛市，之下为熊市

## 实际应用示例

### 场景1：短线交易（分时图）
- 观察价格与MA60的关系
- 价格在MA60上方 → 考虑做多
- 价格在MA60下方 → 考虑做空

### 场景2：波段交易（5m/1h图）
- 观察MA5和MA7的交叉
- 黄金交叉（MA5上穿MA7）→ 买入信号
- 死亡交叉（MA5下穿MA7）→ 卖出信号
- MA99作为止损位参考

### 场景3：趋势判断（1h/1d图）
- 三条均线多头排列（MA5 > MA7 > MA99）→ 强势上涨趋势
- 三条均线空头排列（MA5 < MA7 < MA99）→ 强势下跌趋势
- 均线缠绕 → 震荡行情

## 性能优化
- ✅ 条件绘制：数据点不足时不绘制
- ✅ 变更检测：只在数据变化时更新
- ✅ 异常处理：避免计算错误导致程序崩溃
- ✅ 高效计算：使用pandas的rolling函数（C语言优化）

## 注意事项
1. **初始显示**：所有均线在启动时即可显示（获取了120根历史K线）
2. **实时更新**：均线会随着新数据自动滚动更新
3. **币种切换**：切换币种后所有均线自动重绘
4. **图例显示**：每个图表右上角显示图例，可点击隐藏/显示均线

## 常见问题

### Q: 某条均线不显示？
**A**: 检查数据点数量是否足够（MA99需要至少99个数据点）

### Q: 均线颜色看不清？
**A**: 修改 `plot.py` 中的颜色配置，选择对比度更高的颜色

### Q: 程序启动慢了？
**A**: 计算多条均线会增加CPU负担，可以删除不需要的均线

### Q: 均线更新不及时？
**A**: 检查网络连接状态（右上角有连接指示器），确保WebSocket正常工作
